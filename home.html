.01 Overview of the initial page

Link for the initial website: laosichuan.com

Problems of the website:

  • Usability: Restaurants are exhibited very closely with each other without a clear boundary to separate them.
  • Usability: The menu section is not well organized for audience to read and take actions.
  • Learnability: The navigation bar is too small for audiences to read and understand.
  • Memorability: It is hard to distinguish between different restaurant listed on the website
  • Memorability: There is some duplicate information listed in several sections, leading unnecessary inefficiency for audiences to filtering important information.

Accessibility:

WAVE report points out several errors with the current page. The main errors are missing alternative text and low contrast. Besides, it figures out some redundant link describing the address of each restaurant. No results available on screen-reader

Responsiveness Summary

The responsiveness is achieved by applying “flex” function and specifying changes with “@media” when the screen size becomes smaller. For screen size that is consistent with tablet, the map and the general information becomes vertically consistent instead of the horizontal consistency in the desktop screen mode. In terms of mobile screen size, the number of branches listed in one line is reduced to one. Besides, the font-size increased to 1.2 times for tablet screen and 1.5 times for mobile screen when compared to the desktop screen.
